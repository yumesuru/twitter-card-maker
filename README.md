# Twitter Card Maker

트위터 계정 자기소개 카드를 브라우저에서 만드는 정적 웹툴입니다. 서버 없이 동작합니다.

## 주요 기능

- 고정 캔버스 크기 선택: 1200 x 675 / 1200 x 800
- 입력 내용 실시간 미리보기
- 기본 가로형 / 기본 가로형(확장) 레이아웃 선택
- 브라우저 로컬 저장소 자동 저장
- JSON 저장 및 불러오기
- PNG/JPG 내보내기
- 프로필 이미지 업로드
- 우상단 이미지 0~3개 선택
- 이미지 설명 입력
- 이미지 크롭 위치 직접 드래그 조정
- 이미지 줌 슬라이더 조정
- 성향 태그 ON/OFF 선택
- 중복 태그명 개별 선택 지원
- 카드 색상 커스터마이징
- 폰트 선택
- 모바일 화면 대응
- 미리보기 줌 조정

## 색상 설정

색상 입력은 모두 다음 방식으로 동작합니다.

- 왼쪽 색상 미리보기 상자 클릭: 색상 피커 열기
- 오른쪽 입력칸: HEX 코드 직접 입력

조정 가능한 색상은 다음과 같습니다.

- 성향 태그 강조 색상
- 카드 배경
- 기본 글자
- 닉네임
- 소제목
- 말풍선
- 한마디
- 구분선

## 이미지 크롭

이미지를 업로드한 뒤 크롭 내비게이터 안을 클릭하거나 드래그하면 카드에 보이는 이미지의 초점 위치가 바뀝니다.

확대/축소는 각 이미지 아래의 줌 슬라이더로 조정합니다.

## 폰트

지원 폰트는 다음과 같습니다.

- Asta Sans
- Gowun Batang
- Gowun Dodum
- IBM Plex Sans KR
- Noto Sans KR
- Noto Serif KR
- Pretendard

기본 폰트는 Asta Sans입니다.

## 내보내기

PNG/JPG 내보내기는 `html-to-image`를 사용합니다.

라이브러리는 `vendor/html-to-image.js`에 포함되어 있어 CDN 없이 동작합니다.

## 사용 방법

1. `index.html`을 브라우저에서 엽니다.
2. 오른쪽 입력 폼에서 텍스트, 이미지, 색상, 태그를 수정합니다.
3. 왼쪽 미리보기에서 결과를 확인합니다.
4. 필요한 경우 JSON으로 작업 상태를 저장합니다.
5. 완성 후 PNG 또는 JPG로 저장합니다.
